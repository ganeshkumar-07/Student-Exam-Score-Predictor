## Project Description

The Student Exam Score Predictor is an AI/ML-based application that predicts a student's exam score (0–100) based on academic and lifestyle factors.

The system uses a trained machine learning model to analyze inputs such as study hours, attendance, mental health, sleep duration, and part-time job status.

This project provides both:
- Command Line Interface (CLI) (mandatory)
- Streamlit-based User Interface (UI)

---

## Problem Statement

Many students are unaware of how their daily habits affect academic performance. This project helps predict exam scores and encourages better study habits.

---

## Methodology

- Dataset Source: Kaggle  
- Data Preprocessing: Encoding categorical variables (Yes/No → 1/0)  
- Model Training: Jupyter Notebook  
- Model Saving: best_model.pkl  
- Prediction: CLI + UI
